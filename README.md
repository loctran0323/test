# cursor

Presentation created:

- `amd-presentation.html` - self-contained browser presentation on Acid Mine Drainage (AMD)
- `amd-presentation-draft.pptx` - draft PowerPoint deck that can be uploaded into Google Slides
- `build_amd_pptx.py` - script to regenerate the `.pptx` deck

Usage:

- Open `amd-presentation.html` in a web browser.
- Use the left/right arrow keys or on-screen buttons to navigate slides.
- Print to PDF from the browser if you need a shareable deck file.
- Upload `amd-presentation-draft.pptx` to Google Drive, then open it with Google Slides.
- In Google Slides, you can also use `File -> Import slides` from a new presentation after uploading the `.pptx`.
